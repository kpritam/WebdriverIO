# WebdriverIO
Simple implementation of page object model using 
####WebdriverIO 
####mocha
####chai

##Prerequisites :

###1. Nodejs

###2. WebdriverIO 
          
   npm install webdriverio

###3. Mocha
   
   npm install mocha

###4. Chai

   npm install chai

##How to run :
 
   ./node_modules/.bin/wdio wdio.conf.js
