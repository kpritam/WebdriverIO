# WebdriverIO
Simple implementation of page object model using `WebdriverIO` `mocha` `chai`

## Prerequisites :
  * Nodejs
  * WebdriverIO
```bash
   npm install webdriverio
```
  * Mocha
```bash
   npm install mocha
```
  * Chai
```bash
   npm install chai
```
## How to run :
```bash
   ./node_modules/.bin/wdio wdio.conf.js
```
